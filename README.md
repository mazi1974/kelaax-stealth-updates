# KeLAAX Stealth Updates

Official releases and updates for KeLAAX Stealth traffic automation software.

## Latest Version: V10.0.0

Download the latest version from the [Releases](https://github.com/mazi1974/kelaax-stealth-updates/releases) page.

## Requirements

- **Python 3.11+** - [Download](https://www.python.org/downloads/)
- **Google Chrome** - [Download](https://www.google.com/chrome/)
- Windows 10/11 (64-bit)

## Installation

1. Install Python 3.11+ (check "Add to PATH" during installation)
2. Install Google Chrome
3. Download the installer from Releases
4. Run the installer - dependencies install automatically
5. Launch KeLAAX Stealth and activate with your API key
